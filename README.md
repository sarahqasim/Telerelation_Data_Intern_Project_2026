# Executive Summary  
This project analyses how Zoom, a video communication service experienced a massive surge in user base and revenue during the COVID pandemic in 2020 and how successful it has been in retaining its popularity post pandemic compared to other video communication services. This analysis provides a framework to understand how companies can maintain market retention in the global technology space. 

# Objectives
•	Compare time series pattern for revenue and number of users pre and during the pandemic.
•	Assess how users were retained post pandemic using metrics such as churn rate and net dollar expansion rate (NDER).
•	Compare post pandemic performance between Zoom and Google Meet using factors such as user base, retention rates and customer segmentation.

# Project Deliverables 
•	Scrape financial and user metrics from publicly available financial data into a clean and standardized database.
•	Analyse and uncover key trends and patterns using EDA in Python 
•	Create an interactive dashboard highlighting primary metrics, KPIs and comparative slicers.
•	Document all data sources, findings, insights and actionable recommendations into a report.
•	Deliver final takeaways and understandings in an in-person presentation. 

# Conclusion 
This project aims to collect actionable insights from the successful customer retention of Zoom so that its success may be utilised as an effective framework by other technology companies.
